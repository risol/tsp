## 快速开始

### 安装与运行

```bash
# 克隆项目
git clone <repository-url>
cd tsp

# 启动服务器
deno run --allow-net --allow-read src/main.ts

# 或指定参数
deno run --allow-net --allow-read src/main.ts --root ./www --port 9000 --dev
```

### 第一个页面

在 `www` 目录下创建 `hello.tsx`：

```tsx
import type { PageContext } from "../src/cache.ts";

export default async function (context: PageContext) {
  const { method, query } = context;
  const name = query.name || "World";

  return (
    <html lang="zh-CN">
      <head>
        <meta charset="UTF-8" />
        <title>Hello</title>
      </head>
      <body>
        <h1>Hello, {name}!</h1>
        <p>请求方法: {method}</p>
      </body>
    </html>
  );
}
```

访问 `http://localhost:9000/hello` 即可看到结果。

## 核心概念

TSP-FPM 采用以下核心设计：

- **PageFunction**: 每个 `.tsx` 文件导出的默认函数，负责生成页面内容
- **PageContext**: 传递给页面函数的上下文对象，包含请求的所有信息
- **模块缓存**: 基于文件修改时间的智能缓存，无需重启即可看到代码变更
- **JSX 渲染**: 使用 Preact 渲染 JSX 到 HTML 字符串

## PageContext 详解

`PageContext` 是传递给每个页面函数的核心对象，包含 HTTP 请求的所有信息。

### 类型定义

```typescript
type PageContext = Readonly<{
  method: HttpMethod;
  url: URL;
  headers: Headers;
  query: Record<string, string>;
  body: unknown;
  cookies: Record<string, string>;
  file: string;
  root: string;
}>;
```

### 属性说明

| 属性 | 类型 | 说明 |
|------|------|------|
| `method` | `HttpMethod` | HTTP 请求方法 |
| `url` | `URL` | 完整的请求 URL 对象 |
| `headers` | `Headers` | 请求头对象 |
| `query` | `Record<string, string>` | URL 查询参数 |
| `body` | `unknown` | 请求体数据 |
| `cookies` | `Record<string, string>` | Cookie 数据 |
| `file` | `string` | 当前页面文件路径 |
| `root` | `string` | 文档根目录路径 |

### HttpMethod 类型

```typescript
type HttpMethod = "GET" | "POST" | "PUT" | "PATCH" | "DELETE" | "HEAD" | "OPTIONS";
```

### 使用示例

#### 1. 获取请求方法

```tsx
export default async function (context: PageContext) {
  const { method } = context;

  if (method === "POST") {
    // 处理 POST 请求
  }

  return <div>Method: {method}</div>;
}
```

#### 2. 解析查询参数

```tsx
export default async function (context: PageContext) {
  const { query } = context;

  // URL: /page?name=张三&age=25
  const name = query.name || "匿名";
  const age = parseInt(query.age || "0", 10);

  return (
    <div>
      <p>姓名: {name}</p>
      <p>年龄: {age}</p>
    </div>
  );
}
```

#### 3. 处理 POST 数据

`body` 的值根据 `Content-Type` 自动解析：

| Content-Type | body 类型 |
|--------------|-----------|
| `application/json` | 解析后的对象 |
| `application/x-www-form-urlencoded` | 键值对对象 |
| 其他 | 原始文本字符串 |
| GET/HEAD/DELETE | `null` |

```tsx
export default async function (context: PageContext) {
  const { method, body } = context;

  if (method === "POST" && body) {
    // JSON POST
    if (typeof body === "object" && body !== null) {
      const data = body as Record<string, unknown>;
      const username = data.username as string;
      // ...
    }
  }

  return <form method="POST">
    <input name="username" type="text" />
    <button type="submit">提交</button>
  </form>;
}
```

#### 4. 读取 Cookies

```tsx
export default async function (context: PageContext) {
  const { cookies } = context;

  const sessionId = cookies.sessionId || "未登录";
  const theme = cookies.theme || "light";

  return <div>
    <p>Session: {sessionId}</p>
    <p>Theme: {theme}</p>
  </div>;
}
```

#### 5. 获取请求头

```tsx
export default async function (context: PageContext) {
  const { headers } = context;

  const userAgent = headers.get("user-agent") || "Unknown";
  const accept = headers.get("accept") || "*/*";

  return <div>
    <p>User-Agent: {userAgent}</p>
    <p>Accept: {accept}</p>
  </div>;
}
```

#### 6. 使用 URL 对象

```tsx
export default async function (context: PageContext) {
  const { url } = context;

  return <div>
    <p>完整 URL: {url.href}</p>
    <p>域名: {url.hostname}</p>
    <p>路径: {url.pathname}</p>
    <p>协议: {url.protocol}</p>
    <p>端口: {url.port}</p>
  </div>;
}
```

#### 7. 获取文件信息

```tsx
export default async function (context: PageContext) {
  const { file, root } = context;

  return <div>
    <p>当前文件: {file}</p>
    <p>根目录: {root}</p>
  </div>;
}
```

## 页面开发

### 基本结构

每个 `.tsx` 页面必须导出一个默认函数：

```tsx
import type { PageContext } from "../src/cache.ts";

export default async function (context: PageContext) {
  // 处理逻辑
  return <div>页面内容</div>;
}
```

### 返回 JSX

```tsx
export default async function (context: PageContext) {
  return (
    <html lang="zh-CN">
      <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>页面标题</title>
      </head>
      <body>
        <div id="app">
          <h1>标题</h1>
          <p>内容</p>
        </div>
      </body>
    </html>
  );
}
```

### 使用样式

```tsx
export default async function (context: PageContext) {
  const style = `
    body { font-family: sans-serif; }
    .container { max-width: 800px; margin: 0 auto; }
  `;

  return (
    <html>
      <head>
        <style>{style}</style>
      </head>
      <body>
        <div class="container">
          {/* 内容 */}
        </div>
      </body>
    </html>
  );
}
```

### 条件渲染

```tsx
export default async function (context: PageContext) {
  const { method, query } = context;
  const isDebug = query.debug === "true";

  return (
    <div>
      <h1>页面</h1>
      {isDebug && <div>调试信息</div>}
      {method === "POST" ? <div>已提交</div> : <div>请提交</div>}
    </div>
  );
}
```

### 列表渲染

```tsx
export default async function (context: PageContext) {
  const items = ["Apple", "Banana", "Orange"];

  return (
    <ul>
      {items.map((item, index) => (
        <li key={index}>{item}</li>
      ))}
    </ul>
  );
}
```

### 重定向

页面函数可以返回重定向对象，触发 HTTP 重定向：

```tsx
export default async function (context: PageContext) {
  return { redirect: "/login" };
}
```

指定重定向状态码：

```tsx
export default async function (context: PageContext) {
  return {
    redirect: "/new-location",
    status: 301  // 永久重定向
  };
}
```

可用的状态码：
- `301` - Moved Permanently（永久重定向）
- `302` - Found（临时重定向，默认）
- `303` - See Other
- `307` - Temporary Redirect
- `308` - Permanent Redirect

条件重定向示例：

```tsx
export default async function (context: PageContext) {
  const { cookies, query } = context;

  // 未登录重定向到登录页
  if (!cookies.sessionId) {
    return {
      redirect: `/login?redirect=${query.redirect || context.url.pathname}`
    };
  }

  return <div>已登录内容</div>;
}
```

### 返回自定义 Response

页面函数也可以直接返回 Response 对象：

```tsx
export default async function (context: PageContext) {
  const { query } = context;

  // JSON 响应
  if (query.format === "json") {
    return new Response(JSON.stringify({ message: "Hello" }), {
      status: 200,
      headers: { "Content-Type": "application/json" }
    });
  }

  return <div>HTML 响应</div>;
}
```

## 组件开发

### 创建组件

在 `www/components` 目录下创建组件：

```tsx
// www/components/Header.tsx
interface HeaderProps {
  title: string;
  subtitle?: string;
}

export function Header({ title, subtitle }: HeaderProps) {
  return (
    <header>
      <h1>{title}</h1>
      {subtitle && <p>{subtitle}</p>}
    </header>
  );
}

export default Header;
```

### 使用组件

```tsx
import Header from "./components/Header.tsx";
import Footer from "./components/Footer.tsx";

export default async function (context: PageContext) {
  return (
    <div>
      <Header title="我的页面" subtitle="副标题" />
      {/* 页面内容 */}
      <Footer />
    </div>
  );
}
```

### 传递 PageContext

```tsx
// www/components/Layout.tsx
import type { PageContext } from "../../src/cache.ts";

interface LayoutProps {
  title: string;
  context: PageContext;
  children: unknown;
}

export function Layout({ title, context, children }: LayoutProps) {
  return (
    <html>
      <head>
        <title>{title}</title>
      </head>
      <body>
        <nav>
          <a href="/">首页</a>
          <a href="/about">关于</a>
        </nav>
        {children}
        <footer>
          <p>当前路径: {context.url.pathname}</p>
        </footer>
      </body>
    </html>
  );
}

export default Layout;
```

## 路由规则

### URL 到文件的映射

| 请求路径 | 映射文件 |
|---------|---------|
| `/` | `www/index.tsx` |
| `/about` | `www/about.tsx` |
| `/user/profile` | `www/user/profile.tsx` |
| `/user/` | `www/user/index.tsx` |

### 示例

```
请求路径          文件路径
────────────────────────────────
/               → www/index.tsx
/about          → www/about.tsx
/blog           → www/blog.tsx
/blog/post/123  → www/blog/post/123.tsx
/admin/         → www/admin/index.tsx
```

### 404 处理

如果文件不存在，返回 404 状态码：

```html
404 Not Found
```

### 安全保护

- 禁止访问 `.tsx` 以外的文件
- 禁止路径穿越攻击（`../`）
- 只能访问 `root` 目录下的文件

## 请求处理

### 请求流程

工作流程：
```
HTTP 请求
   ↓
路由解析 → 映射到 .tsx 文件
   ↓
安全检查
   ↓
解析请求参数 (query, body, cookies)
   ↓
构建 PageContext
   ↓
获取页面函数 (缓存或重新加载)
   ↓
执行页面函数 → 返回 JSX
   ↓
渲染 JSX → HTML 字符串
   ↓
返回 HTTP 响应
```

### 错误处理

开发模式 (`--dev`) 会显示详细错误：

```html
<h1>500 Internal Server Error</h1>
<pre>错误消息</pre>
<pre>堆栈跟踪</pre>
```

生产模式只显示通用错误信息。

## API 参考

### PageContext

```typescript
type PageContext = Readonly<{
  method: HttpMethod;
  url: URL;
  headers: Headers;
  query: Record<string, string>;
  body: unknown;
  cookies: Record<string, string>;
  file: string;
  root: string;
}>;
```

### PageFunction

```typescript
type PageFunction = (
  context: PageContext
) => Promise<PageResult> | PageResult;
```

### RedirectResult

用于触发 HTTP 重定向：

```typescript
interface RedirectResult {
  /** 重定向的目标 URL */
  redirect: string;
  /** 重定向状态码，默认 302 */
  status?: 301 | 302 | 303 | 307 | 308;
}
```

### 导出函数

从 `src/cache.ts` 导出：

| 函数 | 说明 |
|------|------|
| `getPage(filepath)` | 获取页面函数（带缓存） |
| `renderJSX(jsx)` | 渲染 JSX 到 HTML |
| `clearCache()` | 清除所有缓存 |
| `getCacheSize()` | 获取缓存数量 |
| `type PageContext` | 页面上下文类型 |
| `type RedirectResult` | 重定向结果类型 |

## 最佳实践

### 1. 始终导出默认函数

```tsx
// 正确
export default async function (context: PageContext) { ... }

// 错误
export async function render(context: PageContext) { ... }
```

### 2. 使用解构赋值

```tsx
export default async function (context: PageContext) {
  const { method, query, url } = context;
  // ...
}
```

### 3. 类型安全

```tsx
export default async function (context: PageContext) {
  const { body } = context;

  if (typeof body === "object" && body !== null) {
    const data = body as { username?: string };
    const username = data.username || "未知";
  }
}
```

### 4. 组件复用

将公共部分提取为组件，减少代码重复。

### 5. 使用 TSX 而非 Template

TSX 文件是页面而非模板，相关的命名应体现这一点。

## 常见问题

### Q: 如何重定向？

A: 返回重定向对象：

```tsx
// 简单重定向（302）
return { redirect: "/target" };

// 永久重定向（301）
return { redirect: "/target", status: 301 };

// 带参数查询的重定向
return { redirect: `/login?redirect=${context.url.pathname}` };
```

### Q: 如何返回 JSON？

A: 直接返回 Response 对象：

```tsx
return new Response(JSON.stringify({ data: "value" }), {
  status: 200,
  headers: { "Content-Type": "application/json" }
});
```

### Q: 如何设置自定义响应头？

A: 通过返回 Response 对象实现：

```tsx
return new Response(html, {
  status: 200,
  headers: {
    "Content-Type": "text/html; charset=utf-8",
    "X-Custom-Header": "value"
  }
});
```

### Q: 如何处理静态资源？

A: 静态资源（图片、CSS、JS）应通过其他服务器提供，或添加相应的路由处理。

### Q: 缓存如何工作？

A: 框架基于文件修改时间缓存页面函数。修改文件后会自动重新加载，无需重启服务器。

### Q: 页面返回值支持哪些类型？

A: 支持以下返回值：
- JSX 元素：渲染为 HTML
- `{ redirect: string }` 对象：触发重定向
- `Response` 对象：直接返回

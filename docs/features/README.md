# TSP 功能特性文档

本文档介绍 TSP 的各种功能特性和使用方法。

## 📚 功能列表

### 核心功能

- [依赖注入](./injection.md) - 类型安全的依赖注入功能
  - 使用 `Page` 包装器注入依赖
  - 完整的 TypeScript 类型支持
  - 线程安全的依赖管理
  - 全局可用，无需 import

- [重定向功能](./redirect.md) - HTTP 重定向支持
  - 返回重定向对象触发 HTTP 重定向
  - 支持多种重定向状态码（301, 302, 303, 307, 308）
  - 简洁的 API 设计

- [自定义响应](./custom-response.md) - 完全控制 HTTP 响应
  - 返回自定义 Response 对象
  - 支持自定义状态码、响应头、内容类型
  - 适用于 API 和特殊场景

- [错误处理](./error-handling.md) - 完善的错误处理机制
  - 开发模式：显示详细错误信息和堆栈跟踪
  - 生产模式：隐藏错误详情，保护敏感信息
  - 支持 try-catch 捕获页面错误

## 💡 使用建议

1. **依赖注入**: 适用于需要在多个页面间共享辅助函数和服务的场景
2. **重定向**: 适用于表单提交后的跳转、登录验证等场景
3. **自定义响应**: 适用于构建 RESTful API
4. **错误处理**: 生产环境务必关闭开发模式，保护敏感信息

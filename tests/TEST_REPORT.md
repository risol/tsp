## 问题汇总

### 已解决的问题 ✅

1. **权限问题** - 添加了 `--allow-write` 权限
2. **响应体泄漏** - 所有响应体现在都被正确消费
3. **子进程清理** - 使用 `cancel()` 正确关闭流

### 待解决的问题 ⚠️

1. **Windows 路径分隔符**
   - 影响: routing_test.ts
   - 原因: Windows 使用 `\` 而非 `/`
   - 解决方案: 调整测试以支持两种路径分隔符

2. **HTML 实体编码**
   - 影响: redirect_test.ts
   - 原因: Preact 自动转义 HTML
   - 解决方案: 调整断言检查解码后的内容

3. **URL 编码处理**
   - 影响: routing_test.ts, redirect_test.ts
   - 原因: UTF-8 编码的字符处理
   - 解决方案: 使用正确的编码断言

4. **文件系统大小写**
   - 影响: routing_test.ts
   - 原因: Windows 文件系统不区分大小写
   - 解决方案: 添加平台特定的断言

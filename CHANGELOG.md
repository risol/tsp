## [0.5.0] - 2026-01-29

### 新增
- **访问日志功能** - HTTP 访问日志记录
  - 支持控制台输出（默认）
  - 支持文件输出（通过配置文件或命令行参数）
  - 标准日志格式：`时间戳 HTTP方法 路径 状态码 "User-Agent"`
  - 配置选项：`accessLogPath` 或 `--access-log` 参数

- **测试用例** - 访问日志测试
  - 8个测试用例覆盖所有日志场景
  - 控制台输出测试
  - 文件输出测试
  - 多种 HTTP 方法和状态码测试

- **文档**
  - 创建开发路线图 (`docs/ROADMAP.md`)
  - 更新配置文档，增加访问日志说明
  - 更新配置示例文件

### 改进
- 更新单元测试运行器，使用 `--allow-all` 权限
- 导出 `logAccess` 函数和 `Config` 接口供测试使用

## [0.4.0] - 2026-01-28

### 新增
- **依赖注入系统重构**
  - 显式依赖声明（不依赖字符串解析）
  - 完整的 TypeScript 类型支持
  - 全局 `Page` 函数，无需 import
  - 支持同步和异步依赖构建
  - 依赖注册/注销管理

- **预编译功能**
  - 生产环境启动时预编译所有 TSX 文件
  - 缓存预热机制
  - 加速首次请求响应

- **开发工具**
  - 预编译工具 (`precompile_tool.ts`)
  - 构建工具 (`build_tool.ts`)
  - 开发模式热重载支持

### 测试
- 添加依赖注入单元测试（10个测试用例）
- 添加访问日志单元测试（8个测试用例）
- E2E 测试改进

### 文档
- 创建预编译文档
- 更新依赖注入文档
- 创建历史文档记录演进过程

## [0.3.0] - 2024-08

### 新增
- 配置文件支持（JSON/JSONC）
- 命令行参数覆盖配置文件
- 自动查找默认配置文件
- 配置优先级：命令行 > 配置文件 > 默认值

### 改进
- 优化启动横幅显示
- 改进帮助信息格式
- 支持注释的配置文件格式

## [0.2.0] - 2024-07

### 新增
- 类型安全的依赖注入功能
- 全局类型声明（types.d.ts）

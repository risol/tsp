# 更新日志

本文档记录 TSP 项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [未发布]

### 新增
- 文档中心结构整合
- 功能文档分类（features/）
- 测试文档分类（testing/）
- 历史文档归档（history/）

### 变更
- 项目名称从 TSP-FPM 更名为 TSP
- 可执行文件名从 tsp-fpm 更名为 tspserver
- 移除所有 PHP-FPM 相关描述和比较

### 文档
- 创建文档中心索引
- 更新所有文档链接
- 添加文档间交叉引用

## [0.3.0] - 2024-08

### 新增
- 配置文件支持（JSON/JSONC）
- 命令行参数覆盖配置文件
- 自动查找默认配置文件
- 配置优先级：命令行 > 配置文件 > 默认值

### 改进
- 优化启动横幅显示
- 改进帮助信息格式
- 支持注释的配置文件格式

## [0.2.0] - 2024-07

### 新增
- 类型安全的依赖注入功能
- 全局类型声明（types.d.ts）
- Page 包装器
- 线程安全的依赖管理

### 改进
- 移除页面文件中的 import 语句
- 自动类型推断
- 更好的 IDE 支持
- Page 函数在全局作用域可用

## [0.1.5] - 2024-06

### 新增
- HTTP 重定向功能
- 自定义 Response 对象支持
- RedirectResult 类型
- 多种重定向状态码（301, 302, 303, 307, 308）

### 修复
- 重定向对象与 VNode 的冲突
- 重定向状态码验证

## [0.1.4] - 2024-05

### 修复
- Windows 平台二进制测试问题
- 文件名处理（.exe 后缀）
- 相对路径和绝对路径问题
- 进程 spawn 失败问题

### 改进
- 跨平台兼容性
- 错误提示信息

## [0.1.3] - 2024-04

### 新增
- 错误处理测试页面
- 生产/开发模式错误显示
- 详细的错误堆栈跟踪（开发模式）

### 改进
- 错误页面样式
- 安全性（生产模式隐藏错误详情）

## [0.1.2] - 2024-03

### 新增
- 自定义 Response 对象功能
- 完全控制 HTTP 响应
- 自定义状态码和响应头
- 不同内容类型支持

### 改进
- API 开发支持
- JSON 响应示例

## [0.1.1] - 2024-02

### 修复
- 模块缓存问题
- 文件修改时间检测
- 开发模式热重载

### 改进
- 缓存性能优化
- @deno/loader 集成
- 二进制编译支持

## [0.1.0] - 2024-01

### 新增
- 初始版本发布
- TSX 页面支持
- 路由解析
- 上下文对象
- 查询参数、POST 数据、Cookies 支持
- 安全路径检查
- Preact JSX 渲染

### 功能
- 基本HTTP服务器
- GET/POST 请求处理
- 表单数据处理
- 模块缓存机制
- 命令行参数支持
